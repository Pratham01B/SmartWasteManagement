# SmartWaste - SIH Project

## Tech Stack
- Backend: Java Spring Boot
- Frontend: React.js + Tailwind CSS
- Database: PostgreSQL (Supabase)
- AI Service: Python FastAPI
- Maps: Leaflet.js

## Coding Rules
- Java mein REST APIs banao
- Har API ke liye proper error handling karo
- Database ke liye JPA/Hibernate use karo
- Comments English mein likho
- Har module ke liye alag package banao

## Project Goal
AI-powered waste management system for 
Smart India Hackathon with modules:
1. Complaint Portal
2. Route Optimizer  
3. Waste Classifier (AI)
4. Analytics Dashboard
5. Worker Management
6. Waste Marketplace

## User Roles
- ADMIN: Full access
- CITIZEN: Complaints + Rewards
- WORKER: Schedule + Routes
- RECYCLER: Marketplace
